# High dynamic range (HDR)
### What is HDR
---
Increasing some range, in this case is light, specifically RGB.

### How to calculate
---
- Basic RGB range : 0 ~ 255, 0 ~ 255, 0 ~ 255
- HDR RGB range : 0 ~, 0 ~, 0 ~

$$
WhatWeSee = BasicRGB = Min(WhiteColorInBasicRGB, hdrRGB)
$$
- Consider as overpowered lights .
